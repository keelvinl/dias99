# Landing Page – Projeto 99 Dias para Emagrecimento

Landing page criada com Next.js + TailwindCSS, pronta para deploy na Vercel.

## 🛠 Tecnologias
- Next.js 14
- TailwindCSS 3
- TypeScript
- GSAP (opcional)

## ▶️ Executar local

```bash
npm install
npm run dev
```

## 🚀 Deploy instantâneo

Basta conectar este repositório ao [Vercel](https://vercel.com/import/git)
